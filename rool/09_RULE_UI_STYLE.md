# UI 风格指南

## 设计关键词
- 大气
- 现代
- 克制
- 企业级
- 微信感的秩序与留白
- 安全感、可信赖感、实时感

## 风格要求
- 大面积留白
- 字体层级清晰
- 圆角卡片
- 轻阴影
- 绿色强调色克制使用
- 不要赛博风，不要炫光，不要复杂拟物

## 布局建议
- 左侧导航 + 顶部状态条 + 主内容区
- 卡片化展示 KPI、告警、流状态
- 详情抽屉或右侧面板承载事件细节
- 矩阵页面优先可读性和状态感，而不是纯视觉铺满

## 视觉参考拆解
参考微信的不是聊天气质，而是：
- 信息秩序
- 间距控制
- 弱化装饰
- 重点信息高可读
- 颜色点到为止
